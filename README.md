# Todo List APP Main (Frontend Mentor/curso)

Este desafio do Frontend Mentor passado pelo curso, foi meu primeiro exercício de React + TailwindCSS, onde tive acompanhamento nas aulas para realiza-lo. Ao final do curso teve a sugestão do professor para que implementássemos o botão de remover itens da lista, eu consegui realizar esta tarefa com sucesso!
